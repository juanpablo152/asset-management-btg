# Frontend

## Correr el proyecto

para correr el proyecto deje 2 opciones.

### primera opción

Para está opción es necesario tener instalado una versión de node > 18 y se hizo con la versión de angular 21.2.2.
Ejecutar el siguiente comando dentro de la carpeta del proyecto para instalar sus dependencias.

```bash
npm install
```
Luego ejecutar el siguiente comando para corre el proyecto.

```bash
npm start
```
Con este comando el proyecto queda expuesto en el puerto 4200. Se puede acceder desde la siguiente url.

http://localhost:4200


### primera opción

Para está opción es necesario tener docker instalado y corriendo, se debe ejecutar el siguiente comando.

```bash
docker compose up --build
```

luego que termine, el proyecto quedara expuerto en la misma ruta que con la primera opción.
Nos damos cuenta que termino cuanto nos indica la url también en la terminal.

http://localhost:4200
